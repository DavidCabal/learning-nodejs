Happy-Birthday-Jo
=================

A happy birthday application for Jo Luijten - http://www.joluijten.com/

## How to run

```sh
$ git clone git@github.com:IonicaBizau/Happy-Birthday-Jo.git hbj
$ cd hbj
$ npm install
$ npm test
```
